# MotionPic AI Directory Submission Checklist

Last updated: 2026-06-03

Use this checklist when submitting MotionPic AI to AI tool directories, product directories, and launch communities. This file is an operator checklist only; do not treat it as proof that a submission has been completed.

## Before Submitting

- [x] Public homepage is live: `https://video.cozyguidehub.com/`
- [x] Public `/health` returns `{"ok":true,"provider":"dashscope"}`.
- [x] `robots.txt`, `sitemap.xml`, `llms.txt`, and `indexnow-key.txt` return 200.
- [x] Homepage, template pages, and guide pages include social metadata for previews.
- [x] Core submission fields are available in `DIRECTORY_SUBMISSION_PACK.md`.
- [x] Search submission checklist is available in `SEARCH_ENGINE_SUBMISSION_CHECKLIST.md`.
- [ ] Support email is final and monitored.
- [ ] At least one clean demo video or screenshot is ready for platforms that ask for media.
- [ ] Decide whether paid directory listing fees are worth it before buying any placement.

## Recommended Submission Order

1. SaaSHub and free/low-cost directories.
2. AI Top Tools and similar free AI tool directories.
3. Toolify only after reviewing whether a paid listing is required or worth it.
4. Futurepedia only after reviewing whether a paid listing is required or worth it.
5. There Is An AI For That after the homepage and at least one example video/screenshot are stable.
6. Indie Hackers and X build-in-public posts after the first search submissions are done.
7. Product Hunt after live-payment readiness and at least one stable demo video.
8. Xiaohongshu/TikTok/YouTube Shorts after recording a clean demo clip.

## Chinese Social Platforms

Checklist:

- [ ] Prepare a clean Chinese version before posting.
- [ ] Do not copy garbled or unverified Chinese text from older files.
- [ ] Record or capture a clean demo clip first.
- [ ] Use a simple hook: one photo becomes a short AI video.
- [ ] Focus on one use case per post: pet, product, old photo, or creator content.

## After Each Submission

- [ ] Record the platform name.
- [ ] Record the submitted URL.
- [ ] Record whether approval is pending, approved, rejected, or paid-only.
- [ ] Record the UTM link used.
- [ ] Check `/admin/analytics` later for channel attribution.
- [ ] Add promising platforms to the weekly growth review.

## Do Not Do Without Confirmation

- [ ] Pay for directory placement.
- [ ] Submit Product Hunt launch.
- [ ] Post to Reddit or communities from the owner's personal account.
- [ ] Use live payment claims before live payment is ready.
- [ ] Claim guaranteed quality or guaranteed no-distortion output.
